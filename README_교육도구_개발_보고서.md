# 2022 개정교육과정 수업도구 개발 보고서

## 프로젝트 개요

본 프로젝트는 2022 개정교육과정의 핵심 역량을 강화하기 위한 교육용 도구 20종(국어과 10종, 수학과 10종)을 개발한 것입니다. 각 도구는 A4 크기의 인쇄 친화적 HTML 형식으로 제작되었으며, 학생들의 자기주도적 학습과 협력 학습을 촉진하도록 설계되었습니다.

---

## 개발 배경 및 목적

### 배경
- 2022 개정교육과정에서 강조하는 핵심 역량(의사소통, 공동체, 비판적 사고, 심미적 감성, 창의적 사고, 미디어 리터러시, 자기관리)의 체계적 함양 필요
- 디지털 교육 환경에서도 활용 가능한 인쇄 기반 학습 자료의 중요성 증대
- 학생 중심의 탐구 및 협력 학습을 지원하는 교수-학습 자료 개발 수요

### 목적
- 2022 개정교육과정 역량 기반의 수업 도구 체계적 개발
- 학생들의 자기주도적 학습 능력 및 협력 학습 역량 강화
- 실생활 연계를 통한 교과 내용의 의미 있는 학습 경험 제공
- A4 인쇄 친화적 디자인으로 현장 적용성 극대화

---

## 교육과정 역량 연계

### 국어과 역량
1. **의사소통**: 대화 작성, 토론 준비, 발표 구성 도구
2. **공동체**: 협력 학습, 토론, 발표 시너지 활동
3. **비판적 사고**: 신문 기사 비판적 읽기, 토론 논리 구성
4. **심미적 감성**: 시 감상 및 창작, 표현 활동
5. **창의적 사고**: 창작 활동, 새로운 표현 방법 모색
6. **미디어 리터러시**: 신문 기사 분석, 미디어 정보 비판
7. **자기관리**: 자기평가, 학습 과정 반성

### 수학과 역량
1. **문제해결**: 문제해결 템플릿, 실생활 응용 문제
2. **추론**: 논리 퍼즐, 규칙 찾기, 패턴 인식
3. **의사소통**: 모둠 협력 활동, 해결 과정 설명
4. **연결**: 일상생활 수학, 다른 교과와의 연계
5. **정보처리**: 데이터 수집 분석, 측정 활동
6. **자기관리**: 경제 생활 수학, 계획 및 반성
